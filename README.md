# frames 框架

## 结构思想

1. 企图把 ui 表现层独立分离
2. 场景通过 main 脚本,加载 Ctl,Ctl 初始化加载 View 和 Modele.

## 核心文件夹

1. script/commponent 公共模块
2. script/data 数据文件夹
3. script/manager 管理器文件夹
4. script/util 工具文件夹

## 核心类

1. PrData
2. PrLayerUtil
3. PrLoadResourceManager
